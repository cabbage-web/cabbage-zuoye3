# 「成长副本」30-60-90 学习路径设计器 — 方案说明

## 一、问题诊断

游戏行业正加速向 AI Native 组织形态演进，员工的核心竞争力从"熟练使用工具"转变为"与 AI 高效协作共创"。然而，传统入职培训体系存在三大痛点：

1. **节奏错配**：制度宣讲与业务介绍周期短（通常 3-5 天），无法支撑 AI 思维的系统性建立
2. **内容断层**：缺乏 AI 工具链的渐进式引入设计，新人往往"一次性接收大量工具"但不知如何组合使用
3. **反馈缺失**：学习过程无追踪、无可视化，HR 无法评估融入效果，新人缺乏成就感

基于以上诊断，本方案以"游戏副本"为隐喻，将为期三个月的学习融入过程设计为**30-60-90 三阶段闯关地图**。

## 二、方案设计

### 2.1 三阶段学习路径

| 阶段 | 主题 | 核心目标 |
|------|------|----------|
| 第一阶段 (0-30天) | 认知觉醒 | 建立 AI 思维，掌握基础工具链，融入组织文化 |
| 第二阶段 (31-60天) | 能力深化 | 深化工具组合应用，建立日常人机协同工作流 |
| 第三阶段 (61-90天) | 价值创造 | 独立驱动 AI 创新项目，沉淀经验为团队资产 |

### 2.2 产品功能架构

- **主页**：Hero 区展示核心理念 + 三阶段卡片 + 预设模板选择
- **路径设计器**：三栏阶段面板，支持模块增删改编辑，AI 工具链可视化
- **进度仪表盘**：圆形进度环 + 技能树 + 阶段进度条，游戏化呈现学习成果
- **导出面板**：一键导出 JSON（数据备份）或打印 PDF

### 2.3 设计理念

采用**赛博朋克 × 游戏副本**视觉风格：深空蓝黑背景搭配霓虹青 / 电光紫 / 琥珀金三色渐变，营造"闯关打怪"的沉浸感。粒子背景动画、发光边框、扫描线叠加等细节强化科技感。

## 三、AI 工具选型与使用理由

开发全程使用 AI 辅助，以下为关键选型：

| 环节 | 工具 | 理由 |
|------|------|------|
| 架构设计 | GPT-4 / Claude | 快速产出 PRD 与技术架构文档的结构化框架 |
| UI 设计 | GPT-4 (视觉建议) | 生成赛博朋克配色方案、动画参数、字体搭配建议 |
| 代码生成 | Cursor / GPT-4 | 生成 React 组件、Zustand 状态管理、Tailwind 样式 |
| 类型定义 | TypeScript 编译器 | 确保全链路类型安全，减少运行时错误 |
| 部署 | Netlify CLI | 一键部署纯前端 SPA，自动 HTTPS，零配置 |

## 四、关键配置

### 4.1 技术栈选择

- **React 18 + TypeScript**：组件化 + 类型安全，游戏行业前端首选
- **Vite**：极速 HMR，开发体验优于 Webpack
- **Tailwind CSS**：原子化样式 + 自定义主题，快速落地设计稿
- **Zustand**：比 Redux 轻量 10 倍，API 直觉化，适合中等复杂度状态
- **Lucide React**：高质量描边图标库，风格统一

### 4.2 数据设计要点

```typescript
// 学习路径 → 阶段数组 → 模块数组 的三层嵌套结构
interface LearningPath { phases: Phase[] }
interface Phase { modules: LearningModule[] }
// 模块包含 category 分类(5种) 和 aiTools 工具映射
interface LearningModule { category: ModuleCategory; aiTools: string[] }
```

内置 3 套预设模板覆盖技术研发、产品策划、美术设计三个核心岗位的 AI 学习路径。

## 六、效果评估

### 6.1 定量指标

- 预设 **3 套岗位模板**（技术/产品/美术），覆盖 80% 游戏公司核心岗位
- 每个模板包含 **13-14 个学习模块**，覆盖 AI 工具 / 人机协作 / 业务理解 / 组织文化 / 实战练习 五大类别
- 型检查零错误，构建产物体积 < 100KB（gzip）

### 6.2 定性评估

- **游戏化设计**：以"副本闯关"隐喻降低 AI 学习的心理门槛
- **可定制性**：HR 可自由增删改模块、调整顺序、修改描述
- **可视化反馈**：实时进度追踪和技能树可视化增强成就感
- **零后端依赖**：纯前端 SPA，部署成本趋近于零

### 6.3 后续迭代方向

1. 接入真实 AI Agent 生成个性化学习建议
2. 支持多人在线协作编辑路径
3. 内置 AI 工具使用引导教程
4. 增加数据面板分析团队整体学习进度
